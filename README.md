# 👤 Random User Generator

A simple Random User Generator built with HTML, CSS, and JavaScript.
This project uses the Random User API to display a new random user with every button click.

## 🚀 Features

- Built with HTML, CSS, and JavaScript
- Uses the Random User API
- Generates a new random user on every button click
- Displays the user's name, email, country, and profile image
- Basic error handling for network issues

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
- Fetch API
